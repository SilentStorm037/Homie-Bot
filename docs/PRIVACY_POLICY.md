# Homie Bot Privacy Policy

Last updated: 17 July 2026

This Privacy Policy explains what data Homie may collect, store, and use when the bot is added to a Discord server or when users interact with Homie commands.

By using Homie, you understand that the bot processes certain Discord data in order to provide its features.

## 1. What Homie Is

Homie is a verified public Discord community bot that provides features such as moderation, welcome messages, achievements, birthdays, leveling, colour roles, reminders, giveaways, Twitch/TikTok live alerts, anonymous venting, starboard, smart replies, and server configuration tools.

## 2. Data Homie May Process

Homie may process Discord data that is needed for enabled features, including:

- Discord server IDs
- Discord channel IDs
- Discord role IDs
- Discord user IDs
- usernames, display names, and avatars shown by Discord
- command interactions
- message content where required for bot features
- moderation actions
- birthday dates saved by users or admins
- achievement progress
- XP and level data
- colour role selections
- warning records
- moderator notes
- reminder text
- giveaway data
- reaction role bindings
- Twitch/TikTok usernames linked by users or admins
- configured welcome, leave, live alert, and notification messages
- ticket, modmail, form, suggestion, appeal, poll, economy, AFK, highlight, feed, and automation data when those features are used

Homie only stores data needed to provide bot functionality.

## 3. Data Stored By Feature

### Server Configuration

Homie stores server settings such as configured channels, roles, welcome settings, starboard settings, automod settings, achievement settings, birthday settings, and live alert settings.

### Moderation

Homie may store warning records, moderator notes, automod infractions, and moderation-related logs depending on server configuration.

### Achievements And Leveling

Homie stores achievement progress, unlocked achievements, XP totals, message activity counters, and related progress data.

### Birthdays

Homie stores birthday dates, optional birth years, privacy settings, and celebration history for the birthday system.

### Twitch And TikTok Alerts

Homie stores linked Twitch/TikTok usernames and optional custom live notification messages/images.

### Anonymous Venting

Homie posts anonymous vent submissions to the configured vent channel. The anonymous vent post does not include the submitter's identity.

Discord still sends interaction data to the bot at runtime so the command can function, but Homie is designed not to include the submitter in the vent message.

### Reminders And Giveaways

Homie stores reminder text, due times, giveaway prizes, winner counts, and related message/channel IDs.

### Smart Memory

If a server enables smart memory, Homie may store short user-saved or admin-saved memory notes. If a server also enables natural memory learning, Homie may save safe, simple preferences detected from normal chat, such as favourite games, nicknames, streaming platform, or stated preferences. Naturally learned memories may include the channel ID where the memory originated so Homie can keep replies relevant to the conversation area.

Smart memory is opt-in, can be viewed or cleared with `/smartmemory` commands, and is designed to skip links, mentions, passwords/tokens, contact details, and sensitive personal categories.

## 4. Message Content

Homie may read message content when needed for features such as:

- automod
- smart replies
- optional smart memory commands
- XP/level tracking
- achievements
- unknown-message learning logs
- message-based moderation
- polls and reaction features

Homie does not sell message content or use it for advertising.

## 5. How Data Is Used

Homie uses stored data to:

- provide server features
- remember server configuration
- perform moderation actions
- track achievements and levels
- send welcome, birthday, and live notification messages
- manage roles and colour roles
- run reminders, giveaways, and starboard
- improve command behavior and reliability

## 6. Data Sharing

Homie does not sell user data.

Homie may interact with third-party services where required for features, such as:

- Discord
- Twitch
- TikTok
- hosting providers

Data may also be visible to server staff or members depending on how a server configures channels, logs, embeds, and public features.

## 7. Data Retention

Homie keeps data for as long as needed to provide enabled features.

Some data may remain until:

- a user removes it with an available command
- a server admin removes it
- the bot owner/operator removes it
- the bot is removed and data deletion is requested

Examples:

- birthdays can be removed with `/birthday remove`
- Twitch links can be removed with `/twitchunlink`
- TikTok links can be removed with `/tiktokunlink`
- reminders can be cancelled with `/remind cancel`
- users can download their server-specific data with `/privacy export`
- users can delete optional stored data with `/privacy delete confirm:DELETE`; moderation-integrity records are pseudonymized and stripped of stored evidence/transcript content

## 8. Data Removal Requests

Server owners may request removal of server-related Homie data.

Users may request removal of personal bot data where reasonably possible.

To request data removal, use the official Homie Support Server: https://discord.gg/RjuSrGXHSa

## 9. Security

Reasonable care is taken to protect Homie's stored data and bot credentials.

However, no system is guaranteed to be perfectly secure. Server owners should only grant Homie the permissions needed for their server.

## 10. Children's Privacy

Homie is intended for use within Discord servers that follow Discord's Terms of Service and age requirements.

Server owners are responsible for running communities in line with Discord's rules and applicable laws.

This Privacy Policy may be updated as Homie changes.

The latest version should be made available through the bot listing, dashboard, support server, or official documentation link.
