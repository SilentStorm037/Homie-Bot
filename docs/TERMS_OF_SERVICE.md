# Homie Bot Terms of Service

Last updated: 5 July 2026

These Terms of Service explain the rules for using Homie, a verified public Discord bot designed for community management, moderation, achievements, welcome messages, live notifications, birthdays, anonymous vents, reminders, giveaways, colour roles, leveling, and other server features.

By adding Homie to a Discord server or using Homie commands, you agree to these terms.

## 1. Who Can Use Homie

Homie is intended for Discord servers and Discord users who follow Discord's Terms of Service, Community Guidelines, and all applicable laws.

If a server uses Homie, the server owner and administrators are responsible for configuring Homie appropriately for that server.

## 2. Server Responsibility

Homie is a configurable bot. Server owners and administrators are responsible for:

- choosing which Homie features are enabled
- setting channels and roles correctly
- managing moderation settings
- informing members about server rules
- using logging, moderation, anonymous venting, and achievement features responsibly
- making sure Homie's permissions are appropriate for the server

Homie does not replace human moderation or server staff judgment.

## 3. Acceptable Use

You may not use Homie to:

- harass, threaten, abuse, or target others
- spam servers, channels, or users
- evade Discord moderation or server moderation
- collect or expose private information about users
- post illegal, hateful, sexual, exploitative, or harmful content
- impersonate others in a harmful or misleading way
- abuse anonymous venting, announcements, moderation tools, or automation features
- violate Discord's Terms of Service or Community Guidelines

Servers or users who abuse Homie may lose access to the bot.

## 4. Moderation Features

Homie includes moderation tools such as warnings, timeouts, kicks, bans, message clearing, automod, mod logs, notes, mass kick tools, and role tools.

These features should only be used by authorized server staff. Server owners are responsible for assigning permissions carefully.

Homie may act based on the configuration set by server administrators, including deleting messages, logging moderation events, assigning roles, or restricting users.

## 5. Anonymous Venting

Homie may offer anonymous venting if enabled by a server.

Anonymous vents are posted without showing the submitter's name in the vent channel. Homie is designed not to include or reveal the submitter in the anonymous post.

However, users should understand that Discord still sends command interaction data to the bot at runtime so the command can function. Server owners should not present anonymous venting as a replacement for emergency support, professional help, or legal confidentiality.

Anonymous venting must not be used to abuse, threaten, harass, dox, or target others.

## 6. Data And Storage

Homie may store server configuration and feature data needed for the bot to work, such as:

- server settings
- configured channel IDs and role IDs
- warning records
- moderation notes
- birthday settings
- achievement progress
- XP and level data
- colour role selections
- Twitch/TikTok links
- reminder and giveaway data
- reaction role bindings
- anonymous vent configuration

Homie only stores data needed for enabled bot features.

Server owners may request removal of server-related bot data through the official Homie Support Server: https://discord.gg/RjuSrGXHSa

No guarantee is made that Homie will always be online, error-free, or available in every server.

Features may be added, removed, renamed, or changed to improve reliability, safety, or maintainability.

- these terms are violated
